# 如何新增一篇知識文章（給有一點程式基礎的你）

整個流程只有 4 步：**複製模板 → 填內容 → 加到列表頁 → 加到 sitemap**。
建議用免費的 [VS Code](https://code.visualstudio.com/) 編輯，記事本也可以。

---

## 第 1 步：複製模板，取一個英文檔名

1. 打開 `blog/` 資料夾
2. 複製 `_template.html`，改名成你的文章檔名，例如：
   - `how-to-choose-invisible-grilles.html`（隱形鐵窗怎麼挑）
   - `balcony-safety-net-guide.html`（陽台安全網指南）

**檔名規則**：全小寫英文 + 連字號 `-`，不要中文、不要空格。
檔名就是網址，跟搜尋主題相關的英文字對 SEO 有幫助。

## 第 2 步：把【】占位符全部換成你的內容

用編輯器開啟新檔案，按 `Ctrl+F` 搜尋 `【`，一個一個替換。共有這幾類：

| 占位符 | 說明 |
|--------|------|
| 【文章標題】 | 出現 3 次（title、og:title、h1），內容要一致 |
| 【一句話描述】 | 出現 3 次，80–120 字，會顯示在 Google 搜尋結果 |
| 【關鍵字1,關鍵字2】 | 3–6 個關鍵字，逗號分隔 |
| 【檔名】 | 出現 2 次，填你第 1 步取的檔名（不含 .html 之後自己會接上） |
| 【日期，格式 2026-08-15】 | 出現 3 次，發文日期 |
| 【分類】 | 例如：選購指南、法規知識、案例分享 |
| 【第一段標題】【內文】 | 文章本體，`<h2>` 是大標、`<p>` 是段落，可自由複製增減 |
| 【問題一】【答案一】 | FAQ 問答，**注意要改兩個地方**（見下方） |

**FAQ 要同步改兩處**：
1. 頁面下方的 `<details>` 區塊（讀者看到的）
2. 頁面上方 `<script type="application/ld+json">` 裡 `"FAQPage"` 那段（給 Google 看的）

兩邊的問題與答案文字要一模一樣。如果不想放 FAQ，把 `FAQPage` 那整段 `<script>...</script>` 和下方 `<section class="blog-faq">...</section>` 都刪掉即可。

檢查方式：全部換完後再搜尋一次 `【`，找不到就代表完成。

## 第 3 步：把文章卡片加到 blog.html 列表頁

打開根目錄的 `blog.html`，找到 `<div class="blog-grid">`，
在裡面**最上方**貼上這段（新文章排最前面），並替換大寫標記：

```html
          <a href="blog/檔名.html" class="blog-card">
            <div class="blog-card-body">
              <div class="blog-meta"><span class="spec-tag">分類</span><time datetime="2026-08-15">2026-08-15</time><span>閱讀 5 分鐘</span></div>
              <h2>文章標題</h2>
              <p>一句話描述（跟文章頁的 description 相同即可）。</p>
              <span class="case-link-hint">閱讀全文 →</span>
            </div>
          </a>
```

## 第 4 步：把網址加進 sitemap.xml

打開根目錄的 `sitemap.xml`，在最後的 `</urlset>` **之前**貼上：

```xml
  <url>
    <loc>https://www.anlongsafety.com.tw/blog/檔名.html</loc>
    <lastmod>2026-08-15</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
  </url>
```

---

## 發布前的本機預覽

在網站資料夾開終端機（VS Code 按 `` Ctrl+` ``），執行：

```
python -m http.server 8000
```

瀏覽器開 `http://localhost:8000/blog.html`，點進你的新文章檢查：
- 標題、內文、FAQ 顯示正常
- 手機寬度也看一下（瀏覽器按 F12 → 手機圖示）

沒問題後，把整個資料夾重新上傳到主機，完成！

## 寫作小抄（SEO / GEO 友善）

- **標題放搜尋問句**：「陽台安全網怎麼選？」比「陽台防護介紹」有效
- **段落裡放具體數字**：材質線徑、承重公斤數、價格區間——AI 搜尋最愛引用有數據的句子
- **內部連結**：內文提到產品或案例時，用 `<a href="../products/....html">` 連過去
- **一個月 1–2 篇就好**，穩定持續比爆量重要

懶得手動做的話，把文章草稿（或只是主題）丟給 Claude，說「照 blog 模板做成新文章」即可。
